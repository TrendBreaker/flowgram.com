Flowgram Requires Javascript.

It appears that you've got Javascript disabled. Please enable Javascript in order to experience Flowgram.

{% ifequal browser 'MSIE' %}
  1. Pull down the "Tools" menu, and select "Internet Options".
  2. Select the "Security" tab.
  3. Click the "Custom Level" button.
  4. Make sure "Active Scripting" is set to "Enable".
  5. Click "OK".
{% endifequal %} {% ifequal browser 'Gecko' %} Firefox
  1. Bring up your Options/Preferences pane. On Windows, look under the "Tools" pull-down menu. On Mac, look under the "Firefox" menu next to the Apple menu.
  2. Select the "Content" tab.
  3. Make sure "Enable Javascript" is checked.
  4. Click "OK".
Safari
  1. Bring up your Preferences pane. On Windows, look under the "Edit" pull-down menu. On Mac, look under the "Safari" menu next to the Apple menu.
  2. Select the "Security" tab.
  3. Make sure "Enable Javascript" is checked.
  4. Close the Preferences pane.
{% endifequal %}

If you have a no-javascript plugin protecting your browser, add Flowgram to your list of allowed sites.
